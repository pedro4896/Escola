# Escola
Sistema Acadêmico
O projeto tem como objetivo a criação de um sistema acadêmico: com os seguintes requisitos:
1 - Interface de Login;
2 - Diário escolar com registro de notas dos discentes;
3 - Funcionalidade de frequência de aulas;
4 - Boletim escolar;
5 - Calendário Acadêmico;
6 - Funcinalidade para impressão de documentos como Histórico escolar e Boletim escolar;
Além de aplicações especificas da coordenação:
1 - Cadastro de discentes;
2 - Cadastro de matérias e séries;
3 - Cadastro de docentes;
4 - Modificações no calendário acadêmico;

Todos os dados obtidos pelo sistema serão armazenados em uma base de dados;

Linguagens de programação utilizadas no projeto: HTML, CSS, JavaScript, PHP, SQL.
